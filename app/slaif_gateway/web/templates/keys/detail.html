Reset Counters

Used counters are reset only after confirmation and an audit reason. Reserved counters normally represent in-flight quota reservations; reset them only as an admin repair action.

Hard quota limits remain unchanged. Usage ledger rows remain historical. Redis rate-limit counters are not reset by this action.

Rotate Key

Generate a replacement gateway key through the existing key service and audit trail. Choose one explicit secret delivery mode; lost keys cannot be resent.

Rotation Policy

Rotation creates a new plaintext key and never shows the old plaintext key. Copy the replacement on the next page; it will not be shown again.

By default, the old key is revoked after rotation. Keeping the old key active temporarily allows both keys to authenticate until an admin revokes the old key.
